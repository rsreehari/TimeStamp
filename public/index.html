Timestamp Microservice

Example Usage

/December 15, 2015 /1450137600

Example Output

{ "unix": 1450137600, "natural": "December 15, 2015" }

Supported Formats

  • Unix timestamp (e.g., 1450137600)
  • Natural language dates (e.g., "December 15, 2015")
  • Various date formats including MM/DD/YYYY, YYYY-MM-DD
